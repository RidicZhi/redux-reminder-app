## How To Run

```
git clone https://github.com/RidicZhi/redux-reminder-app.git
cd react-accounts-app
npm install
npm run
npm start
```

## Technology

* React / Create React App
* Bootstrap4
* Redux
* Middleware(redux-logger, redux-devtools)
* Js Date Processing(moment)
* Cookie
* JavaScript ES6
